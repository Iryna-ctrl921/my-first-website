Посібник з використання семантичних тегів

У цьому розділі наведено основні правила використання нових елементів HTML5.

Як правильно структурувати сторінку?

Кожна сторінка повинна мати один тег <main>, який містить основний зміст. Для логічних розділів використовуйте <section>.

Як додавати приклади коду?

Для коротких вставок використовуйте тег <code>. Наприклад: const x = 10;.

Якщо потрібно вивести багаторядковий код, використовуйте комбінацію <pre> та <code>:


<article>
  <h2>Заголовок статті</h2>
</article>
Робота з медіа-контентом

Для зображень з підписами ми використовуємо структуру:

<figure> + <img> + <figcaption>
